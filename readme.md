

## Install
``` bash
git clone https://github.com/smujiang/SRS_GBM.git
cd SRS_GBM
# create your environment
conda create --name srs_gbm python=3.9 pip
# install dependencies
pip install -r requirements.txt
# create config files and add your own settings
echo  DATA_ROOT = "/your/data/root/directory" > config.py  


```
